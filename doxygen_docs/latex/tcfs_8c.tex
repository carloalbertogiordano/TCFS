Get the size of a file in the TCFS file system. 

This function is called when the {\ttfamily getattr} operation is performed on a file in the TCFS to obtain file attributes.


\begin{DoxyParams}{Parameters}
{\em fuse\+\_\+path} & The path of the encrypted file for which the size is requested. \\
\hline
{\em stbuf} & Buffer to store the file attributes, including the size. \\
\hline
{\em fi} & File information structure provided by FUSE. \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
TCFS\+\_\+\+SUCCESS on success, negative error code on failure.
\end{DoxyReturn}
The {\ttfamily tcfs\+\_\+file\+\_\+size} function is invoked to retrieve the size of a file within the TCFS. It decodes the encrypted file path, translates it into the actual file path on the underlying file system, and then uses the {\ttfamily getattr} function to obtain the file attributes, including the file size.


\begin{DoxyParams}{Parameters}
{\em fuse\+\_\+path} & The path of the encrypted file within the TCFS. \\
\hline
{\em stbuf} & Buffer to store the file attributes, including the file size. \\
\hline
{\em fi} & File information provided by FUSE, which may be used to obtain additional details about the file if needed.\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
true on success. On failure, it returns a negative error code representing the type of error encountered.
\end{DoxyReturn}
\begin{DoxyNote}{Note}

\begin{DoxyItemize}
\item The function is a crucial part of file attribute retrieval, and the size is a fundamental attribute of a file.
\item The correct functioning of this function is essential for providing accurate information about the file size.
\end{DoxyItemize}
\end{DoxyNote}
\begin{DoxyWarning}{Warning}

\begin{DoxyItemize}
\item Ensure that the function correctly translates the encrypted file path into the actual file path on the underlying file system.
